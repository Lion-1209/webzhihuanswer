# 知乎问答助手

一个运行在 Edge 浏览器中的知乎问答助手插件，可自动分析知乎网页端的问题，并生成贴近真人风格的知乎回答，支持一键提交。适合需要快速起稿、补充观点或批量浏览时使用。

## 功能特性

- **悬浮球开关**：右下角悬浮球，不干扰知乎页面布局
- **AI 问答面板**：三标签页设计（问答 / 设置 / 历史）
- **真人风格回答**：口语化、接地气，更像知乎真人回答
- **一键提交**：生成回答后可直接发布到知乎
- **多模型支持**：兼容 OpenAI 接口，支持 DeepSeek、StepFun 等国内模型
- **历史记录**：本地保存问答历史，支持再次提问

## 使用流程

### 1. 加载插件

- 打开 Edge，地址栏输入 `edge://extensions`
- 开启右上角「开发者模式」
- 点击「加载解压缩的扩展」
- 选择本项目目录，或使用 Releases 中的 ZIP 包解压后加载

### 2. 配置模型

- 进入任意知乎问题页
- 点击右下角蓝色悬浮球「💬 AI 问答」
- 切换到「⚙️ 设置」标签页
- 填写三项配置：
  - **Base URL**：API 地址，如 `https://api.deepseek.com` 或 `https://api.stepfun.com`
  - **Model**：模型名称，如 `deepseek-chat` 或 `step-3.7-flash`
  - **API Key**：你的 API 密钥
- 点击「测试连接」，直到提示「连接成功」

### 3. 分析并生成回答

- 回到「💬 问答」标签页
- 点击「分析当前问题」
- 等待生成回答，支持继续追问

### 4. 一键提交

- 确认回答内容后，点击「🚀 一键提交回答」
- 插件会自动填入知乎编辑器并提交

## 注意事项

- 知乎的反爬机制可能会检测自动化操作，首次使用建议先手动确认内容再提交
- 配置信息保存在本地浏览器中，不会上传到任何服务器
- 请遵守知乎社区规范，合理使用 AI 辅助
